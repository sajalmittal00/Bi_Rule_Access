Business Park Rules:

1. Active Customers/Contracts/Clients/Accounts:
   - Main Table: stg_sf_TenancyContract
   - Date Column: Contract_Expiry_Date_Formula__c
   - Category Column: Account.CountryName_of_Origin__c
   - SQL Query:  SELECT COUNT(DISTINCT t.Id) AS Active_Customers
    FROM stg_sf_TenancyContract t
    LEFT JOIN stg_sf_Account a ON t.Tenant__c = a.Id
    WHERE t.Contract_Expiry_Date__c >= DATEADD(MONTH, DATEDIFF(MONTH, 0, GETDATE()), 0)
    AND (t.Surrender_Date__c >= DATEADD(MONTH, DATEDIFF(MONTH, 0, GETDATE()), 0) OR t.Surrender_Date__c IS NULL)
    AND (t.Cancellation_Date__c IS NULL OR YEAR(t.Cancellation_Date__c) = YEAR(GETDATE()))
    AND t.IS_BC_Contract__c = 0
    AND t.Manually_Cancelled__c = 0
    AND LOWER(t.Status__c) IN ('active','approved','expired','in review','renewed','submitted to finance','cancelled')
    AND (LOWER(t.First_Line_Item_Unit_Name__c) LIKE '%op%' OR LOWER(t.First_Line_Item_Unit_Name__c) LIKE '%hq%')
    AND LOWER(t.First_Line_Item_Unit_Name__c) NOT LIKE '%bc%'


2. Deals: 
   - Main Table: stg_sf_TenancyContract
   - Date Column: DWC_Signed_Date__c
   - SQL Query: SELECT COUNT(DISTINCT t.Id) AS NumberOfDeals FROM stg_sf_TenancyContract t
    LEFT JOIN stg_sf_Account  a on t.Tenant__c = a.Id
    LEFT JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
    LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id
    LEFT JOIN stg_sf_InventoryUnit IU on IU.Id=cli.Inventory_Unit__c
    WHERE
      LOWER(cli.Product_Name__c) IN ('hq kiosk','head quarter','office park')
      AND IU.Record_Type_Name__c <> 'Retail'
      AND t.Activated_Date__c IS NOT NULL
      AND t.Manually_Cancelled__c = 0
      AND old.Name IS NULL
      AND YEAR(t.DWC_Signed_Date__c)= YEAR(GETDATE()) and MONTH(t.DWC_Signed_Date__c) <= 12
      AND t.Total_Rent_Price_Formula__c IS NOT NULL


3. Revenue / Overall Revenue / Total Lease Price
    - Main Table: stg_sf_TenancyContract
    - Date Column: DWC_Signed_Date__c 
    - Category Column: cli.Product_Name__c
    - Custom Column: New: Lead_Source__c = "New Client" 
                     All: Lead_Source__c in All
                     Upgrade: Lead_Source__c = "Upgrade Existing client"
                     Downgrade:  Lead_Source__c IN ("Downgrade Existing client", "Downgrade from BP") 
                     New + Upgrade:  Lead_Source__c IN ("New Client", "Upgrade Existing client")

    - Target Output (example): (Overall 2026 → 23M ,2025 → 15M,2024 → 10M)
    - SQL Query: SELECT SUM(GrandTotal) AS Achieved_YTD FROM
            (
            SELECT
                t.Id,
                MAX(t.Total_Rent_Price_Formula__c) AS GrandTotal
            FROM stg_sf_TenancyContract t
            LEFT JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
            LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id 
            LEFT JOIN stg_sf_InventoryUnit IU on IU.Id=cli.Inventory_Unit__c
            WHERE
                    t.Manually_Cancelled__c = 0
                AND t.Activated_Date__c IS NOT NULL
                AND YEAR(t.DWC_Signed_Date__c) = YEAR(CURRENT_TIMESTAMP)
                AND MONTH(t.DWC_Signed_Date__c) <= 12
                AND old.Id IS NULL
                AND IU.Record_Type_Name__c <> 'Retail'
                AND (
                       cli.Product_Name__c IN
                       ('HQ Kiosk','Head Quarter','Office Park')
                    OR cli.Product_Type_Name__c LIKE '%Permanent Desk%'
                    OR cli.Product_Type_Name__c LIKE '%Permanent Office%'
                    OR cli.Product_Type_Name__c LIKE '%Smart Desk%'
                    OR cli.Product_Type_Name__c LIKE '%Smart Office%'
                    OR cli.Product_Type_Name__c LIKE '%Permanent Exclusive Office%'
                    OR cli.Product_Type_Name__c LIKE '%Express Smart Desk%'
                    OR cli.Product_Type_Name__c LIKE '%Standard Smart Desk%'
                )

            GROUP BY
                t.Id
            ) X;


    - RULE: When generating revenue queries that reference a contract table plus any child table (e.g., stg_sf_ContractLineItem), always produce SQL that 
    (1) identifies eligible contracts (filter by product using EXISTS or join), 
    (2) aggregates to a single row per contract (e.g., GROUP BY t.Id and MAX/SUM to compute a contract-level revenue), then 
    (3) aggregates those contract-level rows into the requested period (day/month/quarter/year). Handle edge cases for previous-period calculations (Q1 → prev quarter = Q4 of prev year, month = January → prev month = Dec of prev year). Prefer old.Id IS NULL when excluding renewed contracts.
    

  
4. Occupancy %:
    - Main Table: stg_sf_InventoryUnit
    - Category Column: Building_Number__c
    - SQL Query: SELECT 
    inv.Building_Number__c,
    ROUND(100 * SUM(CASE 
              WHEN LOWER(inv.Product_Type_Name__c) IN ('hq kiosk','head quarter','office park') 
               AND inv.Parent_Unit__c IS NULL 
               AND (LOWER(inv.Availability__c) IN ('leased','dwc usage') OR inv.Is_BC_Inventory__c = 1 OR inv.Availability__c IS NULL) 
               AND LOWER(inv.Record_Type_Name__c) IN ('atm','building','office') 
              THEN inv.Area_in_sq_m__c ELSE 0 END) 
        / NULLIF(SUM(CASE 
              WHEN (inv.Is_BC_Inventory__c = 1 OR inv.Availability__c IS NULL OR LOWER(inv.Availability__c) NOT IN ('blocked','legal case','not available','obsolete')) 
               AND inv.Parent_Unit__c IS NULL 
               AND LOWER(inv.Product_Type_Name__c) IN ('hq kiosk','head quarter','office park') 
               AND LOWER(inv.Record_Type_Name__c) IN ('atm','building','office') 
              THEN inv.Area_in_sq_m__c ELSE 0 END), 0) 
        , 1) AS OccupancyPercent 
FROM stg_sf_InventoryUnit inv   
WHERE inv.Building_Number__c IN ('C1','C2','D1','D2','A1','A2','B2','A4','A3','A5','B4','HQ','B1','B5','B3')
GROUP BY inv.Building_Number__c
ORDER BY OccupancyPercent DESC;
    - Always Show in Percentage

5. Net Leasable Area:
    - Main Table: stg_sf_InventoryUnit
    - Category Column: Building_Number__c
    - SQL Query: SELECT SUM(NetArea) AS NetLeasableArea FROM
    (
    SELECT
        inv.Area_in_sq_m__c AS NetArea
    FROM stg_sf_InventoryUnit inv
    WHERE
        (
            inv.Is_BC_Inventory__c = 1
            OR inv.Availability__c IS NULL
            OR LOWER(inv.Availability__c) NOT IN
               ('blocked','legal case','not available','obsolete')
        )
        AND inv.Parent_Unit__c IS NULL
        AND LOWER(inv.Product_Type_Name__c) IN
            ('hq kiosk','head quarter','office park')
        AND LOWER(inv.Record_Type_Name__c) IN
            ('atm','building','office')
        AND inv.Building_Number__c IN ('C1','C2','D1','D2','A1','A2','B2','A4','A3','A5','B4','HQ','B1','B5','B3')

    UNION ALL

        inv.Area_in_sq_m__c AS NetArea
