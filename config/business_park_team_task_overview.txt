1. Salesperson Revenue / Revenyue for all Salesperson / Revenue YTD Acheived Percentage /Revenue Yearly Acheived Percentage
    - Main Table: stg_sf_TenancyContract
    - Date Column: DWC_Signed_Date__c 
    - Category Column: cli.Product_Name__c
    - Custom Column: New: Lead_Source__c = "New Client" 
                     All: Lead_Source__c in All
                     Upgrade: Lead_Source__c = "Upgrade Existing client"
                     Downgrade:  Lead_Source__c IN ("Downgrade Existing client", "Downgrade from BP") 
                     New + Upgrade:  Lead_Source__c IN ("New Client", "Upgrade Existing client")

    - SQL Query: SELECT 
    SUM(GrandTotal) AS Achieved_YTD,Lead_Owner,SalespersonTarget,
    (SUM(GrandTotal)/SalespersonTarget)*100  as RevenueYearlyTargetAcheievedPercentage,
    (SUM(GrandTotal) / ((SalespersonTarget / 12.0) * MONTH(CURRENT_TIMESTAMP))) * 100 AS RevenueYTDTargetAchievedPercentage
    FROM
    (
        SELECT 
            t.Id,
             COALESCE(t.CS_Team_Member__c, t.Opportunity_Lead_Owner__c, '') as Lead_Owner,
             SYT.Target_Amount__c as SalespersonTarget,
            CASE 
                WHEN COALESCE(t.CS_Team_Member__c, t.Opportunity_Lead_Owner__c, '') IN ('Anna Valeriano', 'Ahmed Maanni') AND LOWER(cli.Product_Type_Name__c) LIKE '%smart%'
                THEN MAX(t.Actual_1st_Year_Rent__c)
                ELSE 
                    MAX(t.Total_Rent_Price_Formula__c)
            END AS GrandTotal
        FROM stg_sf_TenancyContract t
        LEFT JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
        LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id
        LEFT JOIN stg_sf_SalespersonYearlyTarget SYT on SYT.Salesperson_Name__c=Coalesce(t.CS_Team_Member__c,t.Opportunity_Lead_Owner__c,'')
        WHERE 
            t.Manually_Cancelled__c = 0
            AND t.Activated_Date__c IS NOT NULL
            AND t.Renewed_Contract__c IS NULL
            AND old.Id IS NULL
            AND  COALESCE(t.CS_Team_Member__c, t.Opportunity_Lead_Owner__c, '') in 
            (Abbosjon Safarov, Ahmed Maanni, Anna Valeriano, Aristotle Eugenio, Mukti Gurung, Sana Nazarudeen)
            AND YEAR(SYT.From_Date__c)=YEAR(current_timestamp)
            AND YEAR(t.DWC_Signed_Date__c)= YEAR(current_timestamp) and MONTH(t.DWC_Signed_Date__c) <= 12
        GROUP BY 
            t.Id,
            COALESCE(t.CS_Team_Member__c, t.Opportunity_Lead_Owner__c, ''),
            cli.Product_Type_Name__c,
            Target_Amount__c
    ) X
    group by Lead_Owner,SalespersonTarget

3. Pipeline Customers/ Salesperson Pipeline / Offering Stage:
    - Main Table: stg_sf_Opportunity
    - Date Column: CloseDate 
    - SQL Query: SELECT  U.Name AS User_Name,
COUNT(DISTINCT  o.Id) AS PipelineDeals
FROM stg_sf_Opportunity o
LEFT JOIN stg_sf_OpportunityLineItem oli ON o.Id = oli.OpportunityId
LEFT JOIN stg_sf_Product p  ON oli.Product2Id = p.Id
LEFT JOIN stg_sf_User U ON U.Id = o.OwnerId
WHERE 
    (o.StageName NOT IN ('Close Lost','Close Won') OR o.StageName IS NULL)
    AND YEAR(o.CloseDate) = YEAR(GETDATE())
    AND p.Name IN ('HQ Kiosk', 'Head Quarter','Office Park')
GROUP BY 
    U.Name;
