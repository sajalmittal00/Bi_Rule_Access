Business Park & Center Overview Rules:


1. Total Active Customers,Total Active Customers/ Total Active Contracts /Total Active Accounts /Total Active Clients::
    - Purpose: Total Active Customers across Business Park AND Business Center combined
    - SQL Query: SELECT SUM(Active_Customers) AS ActiveCustomers
    FROM
    (
    SELECT COUNT(DISTINCT t.Id) AS Business_Park_Active_Customers
    FROM stg_sf_TenancyContract t
    LEFT JOIN stg_sf_Account a ON t.Tenant__c = a.Id
    WHERE t.Contract_Expiry_Date__c >= DATEADD(MONTH, DATEDIFF(MONTH, 0, GETDATE()), 0)
    AND (t.Surrender_Date__c >= DATEADD(MONTH, DATEDIFF(MONTH, 0, GETDATE()), 0) OR t.Surrender_Date__c IS NULL)
    AND (t.Cancellation_Date__c IS NULL OR YEAR(t.Cancellation_Date__c) = YEAR(GETDATE()))
    AND t.IS_BC_Contract__c = 0
    AND t.Manually_Cancelled__c = 0
    AND LOWER(t.Status__c) IN ('active','approved','expired','in review','renewed','submitted to finance','cancelled')
    AND (LOWER(t.First_Line_Item_Unit_Name__c) LIKE '%op%' OR LOWER(t.First_Line_Item_Unit_Name__c) LIKE '%hq%')
    AND LOWER(t.First_Line_Item_Unit_Name__c) NOT LIKE '%bc%'

    UNION ALL

    SELECT COUNT(DISTINCT t.Id) AS Business_Center_Active_Customers FROM stg_sf_TenancyContract t
    INNER JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
    LEFT JOIN stg_sf_Account a ON t.Tenant__c = a.Id
    WHERE t.Manually_Cancelled__c = 0 AND t.Status__c = 'Active'
    AND (t.Cancellation_Date__c IS NULL OR t.Cancellation_Date__c >= GETDATE())
    AND (
                LOWER(cli.Product_Type_Name__c) LIKE '%permanent desk%'
                OR LOWER(cli.Product_Type_Name__c) LIKE '%permanent office%'
                OR LOWER(cli.Product_Type_Name__c) LIKE '%smart desk%'
                OR LOWER(cli.Product_Type_Name__c) LIKE '%smart office%'
                OR LOWER(cli.Product_Type_Name__c) LIKE '%permanent exclusive office%'
            )
    ) AS X;

                                    SELECT t.Id, 'New Customer' AS LeadGroup
                                    FROM stg_sf_TenancyContract t
                                    INNER JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
                                    LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id
                                    WHERE
                                        LOWER(cli.Product_Name__c) IN ('hq kiosk', 'head quarter', 'office park')  -- DAX only has these 3, no LIKE conditions
                                        AND t.Manually_Cancelled__c = 0
                                        AND t.Activated_Date__c IS NOT NULL
                                        AND old.Name IS NULL

                                        AND t.Lead_Source__c NOT IN ('Downgrade Existing client', 'Upgrade Existing client')
                                    GROUP BY t.Id

                                    UNION ALL

                                    SELECT t.Id, 'Upgrade' AS LeadGroup
                                    FROM stg_sf_TenancyContract t
                                    INNER JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
                                    LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id
                                    WHERE
                                        LOWER(cli.Product_Name__c) IN ('hq kiosk', 'head quarter', 'office park')
                                        AND t.Manually_Cancelled__c = 0
                                        AND t.Activated_Date__c IS NOT NULL
                                        AND old.Name IS NULL
                                        AND t.DWC_Signed_Date__c >= DATEFROMPARTS(YEAR(GETDATE()), 1, 1)
                                        AND t.DWC_Signed_Date__c <= EOMONTH(GETDATE(), 0)
                                        AND t.Lead_Source__c IN ('Upgrade Existing client')
                                    GROUP BY t.Id

                                    UNION ALL

                                    SELECT t.Id, 'Downgrade' AS LeadGroup

                                    INNER JOIN stg_sf_ContractLineItem cli ON t.Id = cli.Contract__c
                                    LEFT JOIN stg_sf_TenancyContract old ON t.Renewed_Contract__c = old.Id

                                        LOWER(cli.Product_Name__c) IN ('hq kiosk', 'head quarter', 'office park')

                                        AND old.Name IS NULL

- Never SUM a revenue column directly on a joined query. Always deduplicate first.
- When calculating any revenue, lease price, rent, or charges — first GROUP BY contract ID and take MAX of the revenue column, then SUM the result in an outer query.
- A contract joined with its line items produces multiple rows per contract. Always collapse to one row per contract before summing any money column.

   - DEFAULT (no explicit time qualifier in user question): WHERE YEAR(Date column) = YEAR(GETDATE())
