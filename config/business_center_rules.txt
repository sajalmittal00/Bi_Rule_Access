1. Active Customers:
    - Table : SF_TenancyContract1_LK_LOAD 
    - SQL Query: SELECT COUNT(DISTINCT t.Id) AS Active_Customers FROM SF_TenancyContract1_LK_LOAD t
                 INNER JOIN SF_ContractLineItem_LK_LOAD cli ON t.Id = cli.Contract__c
                 LEFT JOIN SF_Account_LK_LOAD A ON t.Tenant__c = A.Id
                 WHERE t.Manually_Cancelled__c = 'FALSE'
                     AND t.Status__c = 'Active'
                     AND (t.Cancellation_Date__c IS NULL OR t.Cancellation_Date__c >= GETDATE())

                         LOWER(cli.Product_Type_Name__c) LIKE '%smart office%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent exclusive office%'
                     );
                     
    Rules:
        - Dont Apply Date filters Until Mentioned in the User Question
        - When the question refers to a single product name (like 'Smart Desk'), use '=' instead of LIKE. 
        - When multiple product categories are involved, use 'IN' instead of LIKE

3. Active Customers based on Country:
    - Table : SF_TenancyContract1_LK_LOAD 
    - Date column: SF_TenancyContractPayments_LK_LOAD.Contract_Expiry_Date__c
    - Category Columns: SF_Account_LK_LOAD.CountryName_of_Origin__c (Country Name)
    - SQL Query: SELECT A.CountryName_of_Origin__c AS Country,COUNT(DISTINCT t.Id) AS Number_of_Customers,SUM(t.Actual_1st_Year_Rent__c) AS Actual_1st_Year_Rent FROM SF_TenancyContract1_LK_LOAD t
                 INNER JOIN SF_ContractLineItem_LK_LOAD cli  ON t.Id = cli.Contract__c
                 LEFT JOIN SF_Account_LK_LOAD A ON t.Tenant__c = A.Id
                 WHERE 
                     t.Status__c = 'Active'
                     AND (
                         t.Cancellation_Date__c IS NULL
                         OR t.Cancellation_Date__c >= DATEADD(MONTH, 5, DATEFROMPARTS(YEAR(GETDATE()), 1, 1))
                     )
                     AND t.Contract_Expiry_Date__c >= DATEADD(MONTH, 5, DATEFROMPARTS(YEAR(GETDATE()), 1, 1))
                     AND (
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent desk%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent office%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%smart desk%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%smart office%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent exclusive office%'
                     )
                 GROUP BY A.CountryName_of_Origin__c
                 ORDER BY Actual_1st_Year_Rent DESC

4. Active Customer Year on Year:
    - Table : SF_TenancyContract1_LK_LOAD 
    - Date column: SF_TenancyContractPayments_LK_LOAD.Contract_Expiry_Date__c
    - Category Columns: SF_Account_LK_LOAD.CountryName_of_Origin__c (Country Name)
    - SQL Query: SELECT YEAR(TCP.Due_Date__c) AS Year,COUNT(DISTINCT t.Id) AS Number_of_Customers,SUM(t.Actual_1st_Year_Rent__c) AS Actual_1st_Year_Rent FROM SF_TenancyContract1_LK_LOAD t
                 INNER JOIN SF_ContractLineItem_LK_LOAD cli ON t.Id = cli.Contract__c

                 LEFT JOIN SF_TenancyContractPayments_LK_LOAD TCP ON t.Id = TCP.Tenancy_Contract__c
                 WHERE t.Status__c = 'Active'
                     AND (
                         t.Cancellation_Date__c IS NULL 
                         OR t.Cancellation_Date__c >= DATEADD(MONTH, 5, DATEFROMPARTS(YEAR(GETDATE()), 1, 1))
                     )
                     AND t.Contract_Expiry_Date__c >= DATEADD(MONTH, 5, DATEFROMPARTS(YEAR(GETDATE()), 1, 1))
                     AND (
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent desk%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent office%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%smart desk%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%smart office%' OR
                         LOWER(cli.Product_Type_Name__c) LIKE '%permanent exclusive office%'
                     )
                 GROUP BY YEAR(TCP.Due_Date__c)

10. Inventory Occupany %/ Inventory %, Inventory Customers:
    - Table : SF_TenancyContract1_LK_LOAD 
    - SQL Query: SELECT COALESCE(c.Product_Name, i.Product_Name) AS Product_Name,c.Number_of_Customers AS ActiveCustomers,i.Inventory_Count AS InventoryCustomers,(CAST(c.Number_of_Customers AS FLOAT) / NULLIF(i.Inventory_Count, 0)) * 100 AS InventoryOccupancyPercentage
                 FROM (
                     SELECT 
                         COALESCE(cli.Product_Name__c, 'blanks') AS Product_Name,
                         COUNT(DISTINCT t.Id) AS Number_of_Customers
                     FROM SF_TenancyContract1_LK_LOAD t
                     INNER JOIN SF_ContractLineItem_LK_LOAD cli ON t.Id = cli.Contract__c
                     WHERE cli.Product_Type_Name__c IN ('Express Smart Desk','Permanent Desk','Permanent Office','Permanent Office 2','Smart Desk','Smart Office','Standard Smart Desk')
                       AND t.Status__c = 'Active'
                       AND t.Contract_Expiry_Date__c > GETDATE()
                       AND (t.Cancellation_Date__c >= GETDATE() OR t.Cancellation_Date__c IS NULL)
                     GROUP BY COALESCE(cli.Product_Name__c, 'blanks')
                 ) c
                 FULL OUTER JOIN (
                     SELECT 
                         p.Name AS Product_Name,
                         COUNT(*) AS Inventory_Count
                     FROM SF_InventoryUnit_LK_LOAD1 iu
                     LEFT JOIN SF_Product_LK_LOAD p ON iu.Product_Type__c = p.Id
                     WHERE p.Name IN ('Express Smart Desk','Permanent Desk','Permanent Office','Permanent Office 2','Smart Desk','Smart Office','Standard Smart Desk')
                       AND (iu.Availability__c NOT IN ('Not Available', 'Obsolete') OR iu.Availability__c IS NULL)
                     GROUP BY p.Name
                 ) i
                 ON c.Product_Name = i.Product_Name
                 ORDER BY Product_Name;

    - Rules 
        1. Always show results Productwise
        2. Inventory Names are like (eg: Express Smart Desk,Permanent Desk,Permanent Office,Permanent Office 2,Smart Desk,Smart Office,Standard Smart Desk)
        3. If User Question Mentions Inventory Customer then only mention Inventory Customer vice versa with Active Customer and Inventory Occupany %
